Jack In The Box reveal CSS animation as users scroll down your pages
Easy to install and use, it weight only 2 ko and triggers Animate.css animations

Download JackInTheBox

or Download complete sources on Github (including Coffee Script & Grunt files)

Setup Jack in The Box

1
Link to JQuery 

< script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js" > < /script >

2
Link to the CSS animation library

Link to Animate.css and read the instructions to use it.
(You can link to another CSS animation library by changing JackInTheBox settings)

< link rel="stylesheet" href="http://cdnjs.cloudflare.com/ajax/libs/animate.css/3.0.0/animate.css >

3
Link to JackInTheBox.Js component 

< script src="js/jack-in-the-box.js" > < /script >


Reveal CSS Animations

1
Make an element revealable

Add the CSS class .box to a HTML element: it will be invisible until the user scroll to reveal it.
(You can change this CSS class in the JackInTheBox settings to avoid name conflicts.)

< div class="box">  Content to Reveal Here  < /div >

2
Choose the animation style

Pick an animation style on Animate.css , then add the CSS class to the HTML element

< div class="box slideInLeft" >  Content to Reveal Here  < /div >

3
Activate JackinTheBox

Add this javascript at the bottom of your page, after all the dependencies (Jquery, Animate.css..) 

< script >  
$(function() {
$('body').jackInTheBox();
});
< /script >


You’re done!



Now your animations will be revealed when the user scroll.

Customize Settings

boxClass: Class name that reveals the hidden box when user scrolls.

animateClass: Classs name that triggers the CSS animations (’animated’ by default for the animate.css library)

offset: Define the distance between the bottom of browser viewport and the top of hidden box.
When the user scroll and reach this distance the hidden box is revealed. 

$('body').jackInTheBox({  
boxClass:     'box',      // default
animateClass: 'animated', // default
offset:       100px
});
